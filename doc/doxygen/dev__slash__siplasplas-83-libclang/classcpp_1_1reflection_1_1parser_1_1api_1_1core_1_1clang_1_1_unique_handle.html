template<typename RawHandle, void(*)(RawHandle) Release>
class cpp::reflection::parser::api::core::clang::UniqueHandle< RawHandle, Release >

Provides a iunique-ownership RAII wrapper for raw resources.

This class owns a raw clang resource instanced by a factory function and that must be released through a Release function. This class provides unique ownership of the raw resource.â‡§
